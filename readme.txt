== Description ==

Send the Leadform from Google Ads to your Email 

= Add the URl and code to the Leadform =

1. Generate Code
2. Enter the Code and URL to your Leadform in Google Ads
3. Recivieng the sended Leadform to your choosen Email

== Installation ==

= Using the WordPress Admin Console =
1. Go to the 'Plugins' section of your WordPress Admin Console, and click on 'Add New'.
2. Search for "ASEMT Leadform Hook" and click 'Install Now' to install the plugin.
3. Activate the plugin and navigate to "ASEMT Leadform " from the WP menu.
